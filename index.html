apitrace

What is apitrace?

apitrace consists of a set of tools to:

  • trace OpenGL, OpenGL ES, Direct3D, and DirectDraw APIs calls to a file.
  • retrace OpenGL and OpenGL ES calls from a file
  • inspect OpenGL state at any call while retracing
  • view framebuffers and textures
  • view call data
  • edit trace files
  • profile performance of traces

Download

apitrace is open-source and is hosted on github, the source can be cloned via git.

git clone git://github.com/apitrace/apitrace.git

Binary versions of apitrace are available for some platforms from the github download page.
